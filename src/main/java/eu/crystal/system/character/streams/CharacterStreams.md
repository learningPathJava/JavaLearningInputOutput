# Character Streams
## For handling textual or character based data

- Character Streams are used to perform input and output operations on 16 bit data represented by Unicode or ASCII character set. 
- Character Streams are defined mainly by two abstract classes namely Reader and Writer.

The Reader class defines the character stream input.

The writer class defines the character stream output.

Subclasses
- BufferedReader
    - It is used to handle buffered input stream.
- BufferedWriter
    - It is used to handle buffered output stream.
- FileReader	   
    - It handles the reading of character streams from a file.
- FileWriter
    - It handles the writing of character streams to a file.
